## 📋 About bootstrap-dynamic-themes

bootstrap-dynamic-themes is designed to help you change the look of your Bootstrap 5 website without coding. It offers a simple way to customize colors, fonts, and styles quickly. Unlike Bootswatch, which provides preset themes, this tool lets you create and change themes on the go.

This application is ideal for anyone wanting to make their website visually unique while staying compatible with Bootstrap 5. You do not need any programming experience to use it.

## 🛠️ Using bootstrap-dynamic-themes

Once the app is running, here is how to customize your theme:

1. **Choose a base theme:** Start by selecting one of the built-in Bootstrap 5 styles.

2. **Change colors:** Use the color pickers to adjust primary, secondary, and background colors. You will see changes in real-time.

3. **Adjust fonts:** Pick your preferred font style and size from the available options.

4. **Preview:** The app shows a live preview of how your theme looks on sample pages.

5. **Save your theme:** Click "Save" to store your custom theme as a file on your PC. You can load it again later.

6. **Export CSS:** Export the theme as a CSS file to use directly within your own Bootstrap 5 project.

## ❓ Troubleshooting

- **App does not open:** Make sure your Windows is up to date. Try running the app as administrator (right-click the app shortcut and select "Run as administrator").

- **Changes are not saving:** Verify that you have write permissions in the folder where you are saving.

- **Exported CSS is not working:** Ensure you apply the generated CSS file correctly in your Bootstrap project.

- **Installation issues:** Close any antivirus software temporarily during installation, as it might block the installer.
